# Giri Dontala — Portfolio

A React.js portfolio website with dark/light theme and tech skills marquee.

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation
```bash
npm install
```

### Run locally
```bash
npm run dev
```
Open http://localhost:5173

### Build for production
```bash
npm run build
```

## Features
- Dark / Light theme toggle
- Animated tech skills marquee with real logos
- Experience section with TCS projects
- Resume download button
- Responsive design

## Customization
- Update your info in `src/components/index.jsx`
- Replace resume content in `src/components/Nav.jsx` (downloadResume function)
- Add personal projects in the Experience section

## Tech Stack
- React 18 + Vite
- CSS Variables for theming
- Google Fonts (Syne + DM Sans)
- Devicons CDN for tech logos
